# منصة إدارة المتابعة والاعتماد

نسخة تأسيسية أولية لمنصة مؤسسية عربية RTL، مخصصة للعمل على الجوال والكمبيوتر.

## ما تتضمنه هذه النسخة

- لوحة مؤشرات أولية.
- قائمة تنقل مؤسسية.
- مساحة عمل باسم «تقرير الكفاءة الرقابية».
- بيانات تجريبية فقط.
- تصميم Responsive.
- إعداد جاهز للنشر على GitHub Pages.

## التشغيل محليًا

```bash
npm install
npm run dev
```

## النشر على GitHub Pages

1. ارفع جميع الملفات إلى المستودع.
2. افتح Settings ثم Pages.
3. اختر GitHub Actions كمصدر النشر.
4. عند كل تحديث على فرع `main` سيتم النشر تلقائيًا.

## ملاحظات أمنية

- لا ترفع أي مفاتيح سرية.
- ملف `.env` مستبعد من Git.
- لا تستخدم `service_role` داخل الواجهة.
- ربط Supabase سيتم في مرحلة لاحقة بعد اعتماد الجداول وسياسات RLS.
